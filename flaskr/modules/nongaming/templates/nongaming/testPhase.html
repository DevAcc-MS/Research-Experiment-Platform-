Part #2: Test Phase

There would be 20 tasks in the test phase. In each task you will be taken to a start object position and will be asked to locate the ‘instructed’ target object, followed by virtually walking to the target object.

The information of start & target object, & other instructions would be displayed on screen as shown below:

Landing screen

A landing screen similar to this will be present for each task. Press R when you are ready

Orientation

After pressing R, walls & source object will appear for few seconds to help you orient according to surrounding

Pointing Task

Once the walls disappear, press T to point towards the target object's location

Walk to Target Object

Now, move to the location of target object and press 'C'
