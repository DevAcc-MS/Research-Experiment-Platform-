Short Exploration

Please download the zipped file. Then extract, open & explore the virtual environment below to get familiar with the controls, movement etc of the environment. Once you get accustomed to it, you can close this virtual environment by pressing 'ESC' key. You can explore it as long as you want.

NOTE: For Windows platform, run Practise_Env.exe. For Linux, run LinuxBuild.x86_64 (first you'll need to run following command in terminal: chmod +x LinuxBuild.x86_64)
